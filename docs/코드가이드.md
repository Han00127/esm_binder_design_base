# 코드 가이드 — esm_binder_design_base (논문 Algorithm 11 재현)
읽는 순서: PLAN.md → run.py(드라이버, 전체 흐름) → optimize.py(★Alg11 코어 루프) → losses.py → esmc_prior.py → esmfold_diff.py → rank_all.py. 처음엔 run.py + optimize.py 두 개면 80% 이해.


> 이 문서 하나로 "각 파일이 뭘 하고, 어떻게 연결되며, 어디서부터 읽을지"를 파악.

## 1. 한눈에 — 전체 파이프라인 흐름도

```
 입력: configs/trastuzumab_her2.yaml
   (항원 서열 + VH/VL 서열 + CDR ranges + epitope 위치)
        │
        ▼  ───────────────────  run.py  (드라이버) ───────────────────
        │  · scFv = VH-linker-VL 조립 (scfv.py)
        │  · 복합체 [항원 | scFv] → features (res_type [1,L,33])
        │  · 인덱스 계산: CDR(설계대상) / epitope(target) / scFv(fold)
        │  · prompt_ids(고정서열) / build_soft_full(soft→res_type) 준비
        │
        ▼
 ┌───────────── Phase 1: 생성 (trajectory N회 반복) ─────────────┐
 │  optimize.py : optimize_binder()  = Algorithm 11 메인 루프      │
 │                                                                │
 │   x = logits [L,20]  (init: 고정=10, CDR~N(0,1e-4), Cys=-1e6)   │
 │   for k = 1..K(=150):                                           │
 │     T_k = cosine anneal (1 → 0.01)        ← 점점 sharp          │
 │     soft = softmax(x / T_k)                                     │
 │     ┌─ 구조손실 ───────────────────────────────────────┐       │
 │     │ build_soft_full(soft) → res_type [1,L,33]         │       │
 │     │ esmfold_diff.distogram_forward ──────────────────┼──▶ ESMFold2(base 6B) │
 │     │   → distogram_logits [1,L,L,64]  (미분가능)       │       │
 │     │ losses: inter_contact(CDR↔epitope)               │       │
 │     │       + intra_contact(scFv) + globularity(scFv)   │       │
 │     │ g_struct = ∇x L_struct  (그래프 해제)              │       │
 │     └───────────────────────────────────────────────────┘       │
 │     ┌─ LM prior (자연성) ──────────────────────────────┐       │
 │     │ esmc_prior.ESMCPrior.score(soft[scFv], maskCDR) ─┼──▶ ESMC-600M │
 │     │   → masked pseudo-PPL (낮을수록 자연)             │       │
 │     │ g_LM = ∇x L_LM                                    │       │
 │     └───────────────────────────────────────────────────┘       │
 │     g = norm(g_struct) + 0.05·norm(g_LM)  ;  x -= α_k·g  (SGD)  │
 │     저온(T<0.05): b* = distogram-ipTM proxy 최고 추적            │
 │   → 설계 CDR(argmax) → graft → scFv 후보                         │
 └──────────────────────────────────────────────────────────────┘
        │  후보 N개 (scFv 서열 + disto_iptm)  → runs/cand_g*.json
        ▼
 ┌───────────── Phase 2: 선택 (rank_all.py) ─────────────────────┐
 │  disto_iptm 상위 top-K 선별                                     │
 │  각 후보 × 4 critic:  run_esmfold2 폴딩(msa=auto) ────────────┼──▶ Experimental critic 4종 │
 │    → metrics.py: 인터페이스 ipSAE                              │       │
 │  critic 평균 ipSAE → 랭킹                                       │
 └──────────────────────────────────────────────────────────────┘
        │
        ▼  runs/rank_out/ranked.json  (최고 설계 + avg_ipsae)
```

**한 줄 요약:** *soft 서열을 ESMFold2 distogram으로 미분 최적화(+ESMC 자연성), 여러 후보 생성 →
critic 앙상블 ipSAE로 골라낸다.*

|---|---|---|

## 3. 모듈별 상세 (파일 → 역할 → 핵심 함수 → 입출력)

| 파일 | 역할 | 핵심 함수 | 입력 → 출력 |
|---|---|---|---|

| (복사) model_hooks, run_esmfold2 | 검증된 저수준 | — | ESMFold2 로드/폴딩 |

## 4. 핵심 텐서 흐름 (shape)

```
x (설계 logits)          [L, 20]        ← 최적화 대상 (L=384=항원142+scFv242)
 └ softmax(x/T)          [L, 20]   soft
   └ build_soft_full     [1, L, 33]      res_type 분포 (AA=col 2~21)
     └ ESMFold2 forward  → distogram_logits [1, L, L, 64]   (잔기쌍 거리분포)
       └ losses          → scalar (inter/intra/glob)
 └ (scFv 부분) [242,20]
   └ ESMC-600M           → logits [1, 244, 64]  → pseudo-PPL scalar
```

핵심 매핑: **우리 20-AA 컬럼(j=0..19) = res_type id (j+2)**, ESMC token id는 `ESMC_PERM`(esmc_prior.py).

## 5. Algorithm 11 ↔ 코드 줄 대응 (핵심만)

| 논문 Alg11 | 코드 위치 |
|---|---|
| init logits (line1) | `optimize.init_logits` |
| `T_k = T_min+(1-T_min)·½(1+cos(πk/K))` (line5) | `optimize.py` 루프 |
| `soft = softmax(x/T_k)` (line7) | `optimize.py` `soft_s` |
| distogram only (line11) | `esmfold_diff.distogram_forward` |
| Intra/Inter/Glob (line18-20) | `losses.intra/inter_contact, globularity` |
| MaskedPseudoPPL (line22) | `esmc_prior.score` + `losses.masked_pseudo_ppl` |
| gradient 정규화 (line24-25) | `optimize._norm_grad` |
| SGD `x -= α·g` (line27) | `optimize.py` |
| b* (line14-15) | `optimize.py` 저온 disto_iptm 추적 |
| 후처리 ensemble 랭킹 | `rank_all.py` |
